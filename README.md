# mini-projects
# 💻 Python Mini Projects Collection

## 📌 Project Overview

This repository contains a collection of **Python mini projects** demonstrating core programming concepts such as data structures, object-oriented programming, file handling, and real-world system simulations.

These projects are designed to strengthen problem-solving skills and provide hands-on experience with practical applications.

---

## 🚀 Projects Included

### 📖 1. Dictionary Management System

A simple console-based application to manage words and meanings.

#### 🔹 Features:

* Add new words with meanings
* Search for a word
* Display all words
* Update meanings
* Delete words

#### 🛠️ Concepts Used:

* Dictionaries
* Conditional statements
* Loops

---

### 🚆 2. Railway Ticket Reservation System

A simulation of a railway booking system using Object-Oriented Programming.

#### 🔹 Features:

* View available trains
* Book tickets
* View booking history
* Cancel tickets
* Seat availability management

#### 🛠️ Concepts Used:

* Classes & Objects
* Lists & Dictionaries
* Date handling (`datetime`)

---

### 🛒 3. Supermarket Billing System

A billing system that allows users to add products to cart and generate bills.

#### 🔹 Features:

* Display available products
* Add items to cart
* View cart details
* Generate bill with GST calculation
* Save bills in JSON file
* View previous bills

#### 🛠️ Concepts Used:

* File handling (JSON)
* Data structures (lists & dictionaries)
* Object-Oriented Programming
* Billing logic with tax calculation

---

## 🛠️ Technologies Used

* **Python**
* **JSON**
* **Datetime Module**

## ▶️ How to Run

1. Install Python (if not installed)
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/python-mini-projects.git
   ```
3. Navigate to the project folder:

   ```bash
   cd python-mini-projects
   ```
4. Run any project file:

   ```bash
   python filename.py
   ```
